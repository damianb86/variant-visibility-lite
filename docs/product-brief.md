# Product brief

## Problema
Shopify publishing is product-level, but merchants often need to hide individual variants from shoppers.

## Usuario objetivo
Merchandiser managing discontinued colors, wholesale-only variants, samples or temporary availability.

## Merchant persona
Merchant con catalogo activo, equipo chico y necesidad de resolver un problema operativo puntual sin instalar una suite pesada.

## Flujo principal
Choose variants, mark visibility rules, preview the storefront selector behavior, and keep checkout blocking as a future safeguard.

## Propuesta de valor
A transparent storefront control layer using variant metafields and theme app extension guidance.

## MVP
- Variant rule table with status and reason.
- Selector behavior preview for first visible variant, hidden variants and deep links.
- Settings form for default behavior and checkout safeguard readiness.
- Explicit limitations around theme support.

## Futuro fuera de alcance
- Theme app extension for Dawn and selected OS 2.0 themes.
- Checkout validation Function for hidden variant protection.
- Collection-specific visibility rules.
